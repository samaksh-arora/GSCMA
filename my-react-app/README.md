# GSCMA Website - Global Supply Chain Management Association

A full-stack web application for the Global Supply Chain Management Association at Wayne State University.

## Tech Stack

- **Frontend:** React, TailwindCSS, DaisyUI
- **Backend:** Node.js, Express
- **Database:** MongoDB Atlas
- **Authentication:** Firebase Auth
- **Hosting:** Vercel (frontend), Render/Heroku (backend)

## Setup Instructions

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- MongoDB Atlas account
- Firebase account
